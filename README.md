## 🧠 Project Overview

This project simulates a **mechanical system** and creates its **digital twin**, allowing real-time monitoring and anomaly detection.

The system uses **PCA-based machine learning** to identify abnormal behavior and predict potential failures.

---

## 🔧 Features

- 🔐 Secure Login System  
- 📊 Real-time Dashboard (auto-refresh)  
- 🤖 AI-based Anomaly Detection  
- 📈 Interactive Visualizations (Plotly)  
- 💾 Historical Data Logging (CSV)  
- ⚠️ Health Monitoring & Alerts  
- 📥 Data Export (CSV Download)
